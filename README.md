# Cours de Micro-Frontends / Microservices

Voici le repository pour mon cours de micro-frontends et microservices (EFREI).

Vous pourrez retrouver les différents exercices et checkpoints dans les branches correspondantes de ce projet

- `checkpoint2` — **CP2 : Le Shell** — Mise en place de la structure de base du Shell micro-frontend.
- `checkpoint3` — **CP3 : La Résurrection** — Découpage de l'application en modules et ajout du premier micro-frontend (mfe-header).
- `checkpoint4` — **CP4 : La Babel** — Ajout du module mfe-lobby et mise en place de la communication inter-modules via l'Event Bus.
- `checkpoint5` — **CP5 : Le Catalogue** — Implémentation du micro-frontend mfe-catalog pour afficher les produits.
- `checkpoint6` — **CP6 : Le Panier** — Ajout du micro-frontend mfe-cart pour la gestion du panier.
- `checkpoint7` — **CP7 : L'Assemblage Final** — Intégration finale de tous les micro-frontends et finalisation du projet.
